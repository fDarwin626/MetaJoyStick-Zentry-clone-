## ğŸ› Troubleshooting

**Problem: "Port 5173 is already in use"**
- Solution: Close other apps using that port, or the terminal will suggest another port

**Problem: "Module not found" errors**
- Solution: Delete `node_modules` folder and run `npm install` again

**Problem: Animations not working**
- Solution: Make sure GSAP is installed by running `npm install gsap`
